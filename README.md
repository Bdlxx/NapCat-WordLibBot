# NapCat-WordLibBot

基于 NapCat 的 QQ 词库机器人，支持词库管理、伪人 AI 对话、签到好感度、结婚系统等功能。

---

## 指令

### 词库管理（管理员）

| 指令 | 说明 |
|------|------|
| `羽笙跟我学 关键词 答 回复` | 添加精准匹配词条 |
| `添加模糊词条 关键词 答 回复` | 添加模糊匹配词条 |
| `羽笙跟我学` | 两步式添加词条 |
| `羽笙忘掉 关键词` | 删除关键词及所有回复 |
| `羽笙忘掉 关键词 序号` | 删除指定回复 |
| `羽笙忘掉 序号` | 按序号删除关键词 |
| `羽笙回忆一下` | 查看所有关键词列表 |
| `羽笙回忆一下 关键词` | 查看指定关键词的回复 |

### 签到系统

| 指令 | 说明 |
|------|------|
| `签到` / `羽笙签到` | 每日签到获取好感度 |
| `羽笙好感度` | 查看好感度 |
| `签到排行` | 查看好感度排行榜 |
| `羽笙以后叫我 昵称` | 设置昵称（需 10 好感度） |

### 其他功能

| 指令 | 说明 |
|------|------|
| `羽笙赞我` | 给自己点赞 |
| `转码` | 获取消息的 QQ 原生格式 |

### 伪人模式（主人专用）

| 指令 | 说明 |
|------|------|
| `@羽笙 切换glm` | 切换到 GLM 模型 |
| `@羽笙 切换gemini` | 切换到 Gemini 模型 |
| `@羽笙 当前模型` | 查看当前模型 |
| `@羽笙 清除历史` | 清除当前会话历史 |
| `@羽笙 清除所有历史` | 清除所有历史 |

## 变量

### 文本变量

| 变量 | 说明 |
|------|------|
| `[qq]` / `[user_id]` | 触发者 QQ 号 |
| `[name]` / `[nickname]` | 触发者昵称 |
| `[nick]` | 自定义昵称（无则用原昵称） |
| `[card]` | 群名片 |
| `[group]` / `[group_id]` | 群号 |
| `[favor]` | 当前好感度 |
| `[time]` | 当前时间 |
| `[date]` | 当前日期 |
| `[datetime]` | 日期时间 |
| `[bot_id]` | 机器人 QQ 号 |
| `[message_id]` | 消息 ID |
| `[raw_message]` | 原始消息内容 |

### 随机数变量

| 变量 | 说明 |
|------|------|
| `[r1-100]` | 1-100 随机数 |
| `[r1-1000]` | 1-1000 随机数 |
| `[rX-Y]` | X-Y 随机数（动态） |

### 特殊标记

| 标记 | 说明 |
|------|------|
| `[@qq]` / `[@QQ]` | @触发者 |
| `[@[QQ]]` | @触发者（兼容格式） |
| `[img:URL]` | 发送图片 |
| `[avatar]` | 发送触发者头像 |
| `[next]` | 分段发送消息 |

### QQ 原生格式

| 格式 | 说明 |
|------|------|
| `[pic={UUID}.jpg]` | 图片（QQ 缓存） |
| `[pic=URL]` | 图片（网络链接） |
| `[FaceXXX.gif]` | QQ 表情 |
| `[emoji=十六进制]` | Emoji 表情 |
| `[at=QQ号]` | @指定用户 |
| `[at=all]` | @全体成员 |
| `[Voi={UUID}.amr]` | 语音消息 |
