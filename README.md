# 羽笙 QQ 机器人

基于 NapCat 的 QQ 机器人项目，支持词库管理、伪人模式、签到系统等功能。

## 功能列表

- 📝 **词库系统** - 自定义关键词回复，支持精准/模糊匹配
- 🤖 **伪人模式** - AI 智能回复，支持双模型切换
- 💕 **签到系统** - 每日签到、好感度、排行榜
- 💒 **结婚系统** - 群内结婚功能
- 🔄 **转码功能** - 获取图片、表情等的 QQ 原生格式

## 词库变量

### 文本变量

| 变量 | 说明 | 示例 |
|------|------|------|
| `[qq]` / `[user_id]` | 触发者 QQ 号 | `123456789` |
| `[name]` / `[nickname]` | 触发者昵称 | `小明` |
| `[nick]` | 自定义昵称（无则用原昵称） | `笨蛋` |
| `[card]` | 群名片 | `群主大人` |
| `[group]` / `[group_id]` | 群号 | `987654321` |
| `[favor]` | 当前好感度 | `15` |
| `[time]` | 当前时间 | `14:30:25` |
| `[date]` | 当前日期 | `2026-04-29` |
| `[datetime]` | 日期时间 | `2026-04-29 14:30:25` |
| `[bot_id]` | 机器人 QQ 号 | `740979632` |
| `[message_id]` | 消息 ID | `abc123` |
| `[raw_message]` | 原始消息内容 | `你好` |

### 随机数变量

| 变量 | 说明 | 示例 |
|------|------|------|
| `[r1-100]` | 1-100 随机数 | `42` |
| `[r1-1000]` | 1-1000 随机数 | `789` |
| `[rX-Y]` | X-Y 随机数（动态） | `[r0-10]` → `5` |

### 特殊标记

| 标记 | 功能 | 示例 |
|------|------|------|
| `[@qq]` / `[@QQ]` | @触发者 | `[@qq]你好呀` |
| `[@[QQ]]` | @触发者（兼容格式） | `[@[QQ]]来玩吧` |
| `[img:URL]` | 发送图片 | `[img:https://example.com/pic.jpg]` |
| `[avatar]` | 发送触发者头像 | `[avatar]这是你的头像` |
| `[next]` | 分段发送消息 | `第一句[next]第二句` |

### QQ 原生格式（透传）

| 格式 | 说明 | 示例 |
|------|------|------|
| `[pic={UUID}.jpg]` | 图片（QQ 缓存） | `[pic={6E6FAE37-...}.jpg]` |
| `[pic=URL]` | 图片（网络链接） | `[pic=http://gchat.qpic.cn/...]` |
| `[FaceXXX.gif]` | QQ 表情 | `[Face178.gif]` |
| `[emoji=十六进制]` | Emoji 表情 | `[emoji=F09F9180]` |
| `[at=QQ号]` | @指定用户 | `[at=123456789]` |
| `[at=all]` | @全体成员 | `[at=all]` |
| `[Voi={UUID}.amr]` | 语音消息 | `[Voi={61CB20F4-...}.amr]` |

## 命令列表

### 词库管理（管理员）

| 命令 | 说明 |
|------|------|
| `羽笙跟我学 关键词 答 回复` | 添加精准匹配词条 |
| `添加模糊词条 关键词 答 回复` | 添加模糊匹配词条 |
| `羽笙跟我学` | 两步式添加词条 |
| `羽笙忘掉 关键词` | 删除关键词及所有回复 |
| `羽笙忘掉 关键词 序号` | 删除指定回复 |
| `羽笙忘掉 序号` | 按序号删除关键词 |
| `羽笙回忆一下` | 查看所有关键词列表 |
| `羽笙回忆一下 关键词` | 查看指定关键词的回复 |

### 签到系统

| 命令 | 说明 |
|------|------|
| `签到` / `羽笙签到` | 每日签到获取好感度 |
| `羽笙好感度` | 查看好感度 |
| `签到排行` | 查看好感度排行榜 |
| `羽笙以后叫我 昵称` | 设置昵称（需 10 好感度） |

### 其他功能

| 命令 | 说明 |
|------|------|
| `羽笙赞我` | 给自己点赞 |
| `转码` | 获取消息的 QQ 原生格式 |

### 伪人模式（主人专用）

| 命令 | 说明 |
|------|------|
| `@羽笙 切换glm` | 切换到 GLM 模型 |
| `@羽笙 切换gemini` | 切换到 Gemini 模型 |
| `@羽笙 当前模型` | 查看当前模型 |
| `@羽笙 清除历史` | 清除当前会话历史 |
| `@羽笙 清除所有历史` | 清除所有历史 |

## 部署说明

1. 安装依赖：`pip install -r requirements.txt`
2. 配置 `config.json`（机器人 QQ、主人 QQ 等）
3. 运行：`python main.py`
